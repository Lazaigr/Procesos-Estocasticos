# Procesos Estocásticos

Repositorio con trabajos y proyectos realizados durante el curso de **Procesos Estocásticos**.

## Contenido

- **Repositorio Sergio**: Proyectos colaborativos y plantillas de métodos numéricos
- **Documentos**: Apuntes, resúmenes y materiales de referencia del curso
- **Proyectos**: Implementaciones de conceptos clave como movimiento browniano y simulaciones estocásticas

## Tecnología

Este repositorio utiliza **Python** para la implementación de algoritmos y simulaciones relacionadas con procesos estocásticos.
